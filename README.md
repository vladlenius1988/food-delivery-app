# Retail Food Delivery App Demo (React)

## Project Overview
This is a React web application for a retail shop, where users can browse products, add them to a shopping cart, and place orders. The app includes a variety of products — from groceries to personal care items.

## Features
- Product listing with images, prices, and categories
- Shopping cart with quantity management (+/-) and delete functionality
- Order form with Email, Phone, and Address fields
- Total price calculation with proper formatting
- Navigation between main page and cart using React Router

## Installation
1. Clone the repository:
```bash
git clone https://github.com/vladlenius1988/food-delivery-app.git
```

### Backend
1. Navigate to backend folder:
```bash
cd backend
```

2. Install dependencies:
```bash
npm install
```

3. Start Server:
```bash
npm start
```

### Frontend
1. Navigate to frontend folder
```bash
cd frontend
```

2. Install dependencies:
```bash
npm install
```

3. Start development server:
```bash
npm run dev
```

#### Usage

Open your browser at http://localhost:3000 (frontend)
Browse products, add them to the cart, and submit an order form
The backend handles order submission
